= Introduction to SQL connector framework

This section provides additional information about the SQL Connector Framework.

== What is the SQL Connector Framework?

The SQL Connector Framework is a ConnId connector framework designed to provide a flexible and extensible base for SQL-based connectors, implementing common reusable functionality for JDBC connections and database operations.

== Key features

* Auto-discovery of database schema from JDBC metadata
* SQL dialect abstraction (PostgreSQL, MySQL, Oracle, SQLite)
* Connection pooling with HikariCP
* Groovy-based configuration DSL
* Automatic generation of CRUD operations
